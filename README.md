# 🚀 Cyber Nexus – AI Landing Page

A modern, responsive cyberpunk-themed landing page built using HTML, CSS, and JavaScript. The project showcases a futuristic AI interface with a clean UI, glowing design elements, and responsive layouts.

## ✨ Features

- Responsive Design
- Cyberpunk UI/UX
- Hero Section
- Features Section
- About Section
- Statistics Section
- Contact Form
- Footer with Social Links
- Smooth Navigation
- Interactive JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it using Live Server in VS Code.
